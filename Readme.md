# Dungeon Editor for Balagur Fate 3

Редактор подземелья для RPG игры с поддержкой NPC управления и боевой системы.

## Особенности

- Добавление NPC (Медведи, Оборотни, Разбойники)
- Сохранение/загрузка в файл
- Боевая система с настраиваемой дальностью
- Логирование событий (консоль + файл)

## Сборка

mkdir build
cd build
cmake ..
make
./dungeon_editor
./tests
